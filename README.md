# node项目

## 项目依赖
1. node框架：**koa**
2. 路由：**koa-router**
3. 热重启：**nodemon**
4. 连接mysql数据库：**mariadb、mysql2、pg、pg-hstore、sequelize、sqlite3、tedious**
5. 跨域：**koa2-cors**
6. 网络请求
7. 进程管理：**PM2**
8. 支持es6语法
9. 打包压缩
10. eslint
11. typescript开发
12. 测试

## 项目结构
1. error
2. util
3. dal
4. service
